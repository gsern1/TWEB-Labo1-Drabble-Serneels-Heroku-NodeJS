Visualize Github data easily

With the TWEB lab webapp, you can easily visualize data taken from the Github API.

Why you should choose the TWEB lab webapp

It is fast, data is taken directly from the Github API.

It is beautiful. We use free JQuery libraries to display the data in easily readable graphs

It is ad free and opensource.

It is responsive and can be viewed on any devices.
